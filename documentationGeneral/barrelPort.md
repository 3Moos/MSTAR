# tldr
### for the three prong<br>
 - one prong for power in
 - one prong for ground
 - one prong to determine if something is connected, like a switch
### for the four prong<br>
 - one prong for power in
 - one prong for ground
 - one prong to determine if something is connected, like a switch
 - **ASSUMABLY** reference ground, can be soldered to ground as seen in one of the boardshttps://old.reddit.com/r/diyelectronics/comments/o620mb/why_does_this_surface_mount_barrel_plug_have/
